# Unity Starter

A repo with handy stuff for Unity projects.
- Color Palette
- Camera Hotkeys
- Editor Shortcuts
- git stuff
- C# editor stuff

Copy git and editor configs to project root and prepend ".":
- editorconfig
- gitattributes
- gitignore
